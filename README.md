# automatisation-ventes
Projet Python d’analyse automatique des ventes : calcul du CA, remises, TVA et génération de fichiers de résultats.

##  Description du projet

Ce projet consiste à automatiser l’analyse des ventes d’une entreprise de e-commerce à l’aide de Python.

L’objectif est de remplacer le travail manuel sur Excel par un script Python capable de :
- traiter des fichiers CSV
- effectuer des calculs automatiques
- générer des résultats structurés
- identifier les meilleures performances

---

##  Prérequis

Avant d’exécuter ce projet, il faut installer :

- Python 3.x
- VS Code
- Matplotlib (pour les graphiques)
